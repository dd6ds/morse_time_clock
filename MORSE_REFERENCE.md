## Timing Rules

### Basic Timing (based on dot duration = 1 unit)
- **Dot**: 1 unit
- **Dash**: 3 units
- **Gap between dots/dashes**: 1 unit
- **Gap between letters**: 3 units
- **Gap between words**: 7 units

### Standard WPM (Words Per Minute)
- **1 WPM**: Dot = 1200ms
- **5 WPM**: Dot = 240ms
- **12 WPM**: Dot = 100ms (used in this app)
- **20 WPM**: Dot = 60ms
- **30 WPM**: Dot = 40ms

## Prosigns (Procedural Signals)
These are sent as single characters without gaps:

- **AA·**: New line
- **AR: ·−·−·** (end of message)
- **AS: ·−···** (wait)
- **BK: −···−·−** (break)
- **BT: −···−** (pause/separator)
- **CL: −·−·····** (closing/going off air)
- **CT: −·−··−** (start copying)
- **KN: −··−·** (go ahead, specific station)
- **SK: ···−·−** (end of contact)
- **SN: ···−·** (understood)
- **SOS: ···−−−···** (distress signal)

## Memory Aids

### Using Sound Patterns
- **A**: ·− = "di-DAH" (alpha)
- **B**: −··· = "DAH-di-di-dit" (Bonaparte)
- **C**: −·−· = "DAH-di-DAH-dit" (Coca-Cola)
- **M**: −− = "DAH-DAH" (mama)
- **O**: −−− = "DAH-DAH-DAH" (oh-my-god)
- **S**: ··· = "di-di-dit" (Si-si-si)

### Common Words in Morse
- **SOS**: ··· −−− ··· (III DDD III)
- **CQ**: −·−· −−·− (DIDI DDID) - calling any station
- **73**: −−··· ···−− (DDIII IIIDD) - best regards
- **88**: −−−·· −−−·· (DDDII DDDII) - love and kisses

## Practice Tips
1. Start slow (5-8 WPM)
2. Learn by sound, not by counting
3. Practice receiving before sending
4. Use the Farnsworth method (faster characters, slower spacing)
5. Learn common words as whole patterns

## Ham Radio Q-Codes (Common)
- **QRL**: Frequency is busy
- **QRM**: Man-made interference
- **QRN**: Natural interference
- **QRO**: Increase power
- **QRP**: Decrease power / low power operation
- **QRS**: Send slower
- **QRT**: Stop transmitting
- **QRX**: Stand by
- **QRZ**: Who is calling me?
- **QSB**: Signal fading
- **QSL**: Acknowledge receipt
- **QSO**: Contact/conversation
- **QSY**: Change frequency
- **QTH**: Location
